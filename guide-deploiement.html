1
Déployer sur Vercel Gratuit ⏱ 10 min
Créer un compte Vercel gratuit
Connexion avec GitHub recommandée (facilite les déploiements futurs)
→ Ouvrir vercel.com/signup
💡 Clique sur "Continue with GitHub" — c'est la méthode la plus rapide et elle te permettra de déployer automatiquement à chaque modification future.

Si tu n'as pas GitHub, crée un compte en 2 minutes d'abord (gratuit).

Créer un repository GitHub "reputia"
C'est là que tu déposeras les fichiers du site
→ Créer le repository

Paramètres recommandés :

  • Repository name : reputia
  • Visibility : Private (ton code reste privé)
  • Cocher "Add a README file"
  • Cliquer "Create repository"
Uploader les fichiers du site
index.html, vercel.json, robots.txt, sitemap.xml, 404.html

Dans ton repository GitHub, clique sur "uploading an existing file" (ou le bouton "Add file" → "Upload files").

Glisse-dépose les 5 fichiers fournis dans le ZIP :

# Fichiers à uploader dans GitHub reputia/ ├── index.html # Le site complet ├── vercel.json # Config déploiement ├── robots.txt # Instructions pour Google ├── sitemap.xml # Plan du site └── 404.html # Page erreur

Clique "Commit changes" pour valider l'upload.

Importer le projet dans Vercel
Vercel détecte automatiquement la config et déploie
→ Nouveau projet Vercel
  1. Clique "Import Git Repository"
  2. Sélectionne ton repo reputia
  3. Laisse tous les paramètres par défaut
  4. Clique "Deploy"
🚀 En 30 secondes, ton site est live sur une URL du type reputia-xyz.vercel.app. Tu peux déjà le partager et le tester !
2
Acheter le domaine ~10 €/an ⏱ 5 min
Choisir et acheter le domaine
Recommandé : reputia.fr ou reputia.io

Vérifie la disponibilité et achète directement :

→ OVH (recommandé · FR) Namecheap (international)
⚠️ Préfère le .fr pour cibler la France — meilleur SEO local. Si reputia.fr est pris, essaie reputia.io, reputia.app ou app-reputia.fr.

Prix indicatif : .fr ≈ 7€/an · .io ≈ 35€/an · .app ≈ 15€/an

3
Connecter le domaine à Vercel (DNS) Gratuit ⏱ 10 min + propagation
Ajouter le domaine dans Vercel
Vercel génère les enregistrements DNS à copier
  1. Dans Vercel, ouvre ton projet reputia
  2. Va dans Settings → Domains
  3. Entre reputia.fr → "Add"
  4. Vercel t'affiche 2 enregistrements DNS à copier

Tu verras quelque chose comme :

# Enregistrements DNS fournis par Vercel Type A : @ → 76.76.21.21 Type CNAME: www → cname.vercel-dns.com
Coller les DNS chez OVH (ou ton registrar)
Dans la zone DNS de ton domaine

Chez OVH :

  1. Va dans Web Cloud → Domaines → Zone DNS
  2. Clique "Ajouter une entrée"
  3. Ajoute l'entrée A : cible 76.76.21.21
  4. Ajoute l'entrée CNAME : sous-domaine www, cible cname.vercel-dns.com
  5. Valide
⏳ La propagation DNS prend 5 à 30 minutes en général (jusqu'à 48h max). Vercel affiche un ✓ vert quand c'est actif. Le certificat HTTPS est généré automatiquement — rien à faire.
4
Brancher les analytics Gratuit ⏱ 5 min
Activer Vercel Analytics (inclus gratuit)
Pour voir tes visiteurs sans RGPD complexe
  1. Dans ton projet Vercel → onglet "Analytics"
  2. Clique "Enable Analytics"
  3. C'est tout — les stats apparaissent automatiquement
📊 Tu verras : nombre de visiteurs, pages vues, pays, sources de trafic — sans cookie banner obligatoire (privacy-first by design).

Optionnel : ajoute aussi Google Search Console (gratuit) pour suivre ton référencement.
